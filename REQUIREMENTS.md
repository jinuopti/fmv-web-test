[개요]
FMV 게임을 웹사이트에서 웹게임으로 개발하고 플레이한다.
동영상 위주로 재생되며 동영상위에 UI 오버레이 하고 선택지 분기, 사물 터치 이벤트 등 기본 FMV 게임 요소를 개발한다.
PC 에서는 브라우저 내 영화관 같은 스크린 UI 에서 플레이 하며 최대화면 확대가 가능하다.
모바일은 플레이시 기본 가로모드 전체화면으로 플레이한다.
초기 MVP 에서는 mp4 기본 파일 재생을 기반으로 하지만 상용 서비스에는 HLS 로 플레이한다.
게임 요소는 기본 UI, 선택지 분기, 특정 사물 터치 가능한 UI 와 터치했을때 분기 혹은 UI 표시 등
보안 요소는 Signed Token 사용.

[기술스택]

- 프런트: React + TypeScript + Vite
- 영상 재생: 기본 <video> + HLS.js
- 오버레이/UI: React DOM + CSS, 고급 애니메이션/핫스팟은 PixiJS v8
- 분기/게임 상태: XState
- 사운드 효과: Howler.js
- 영상 인코딩/배포: Cloudflare Stream + R2
- DB, API, 회원관리, JWT 등 서버 및 DB: 자체 Go 서버 있음

[PixiJS]

- 영상 위에 반짝이는 조사 포인트
- 여러 개의 움직이는 핫스팟
- 확대/축소/패닝
- 애니메이션 강조선, 파티클, 글로우
- 프레임마다 위치가 조금씩 바뀌는 인터랙션 오브젝트

[테스트시나리오]
videos/ 테스트 영상 위치
intro -> run (선택지 영상 loop) -> leave/look 선택하여 분기 재생
일반영상 intro 재생후 선택지 전용 loop 영상 run 재생, 선택지 ui 오버레이, 선택하면 각 영상으로 분기
